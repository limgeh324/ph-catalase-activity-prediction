# 카탈레이스 효소 활성 예측 모델

## 프로젝트 개요

본 프로젝트는 감자 속 카탈레이스(catalase) 효소와 과산화수소의 반응 실험 데이터를 이용하여 pH와 효소 활성의 관계를 분석하고, 머신러닝 기반 회귀 모델을 통해 효소 활성을 예측하는 것을 목표로 한다.

실험 데이터의 개수가 매우 적었기 때문에, 보간(interpolation)과 Monte Carlo 기반 노이즈 시뮬레이션을 활용하여 가상 데이터를 생성하였으며, 이를 바탕으로 Polynomial Regression 모델을 학습시켰다.

---

## 연구 배경

카탈레이스는 과산화수소(H₂O₂)를 물(H₂O)과 산소(O₂)로 분해하는 효소이다.

반응식:

```text
2H₂O₂ → 2H₂O + O₂
```

효소 활성은 pH와 같은 환경 조건에 따라 변화하며, 일반적으로 특정 pH 범위에서 가장 높은 활성을 보인다.

본 프로젝트에서는 pH 변화에 따른 카탈레이스 활성 변화를 분석하고, 이를 데이터 기반으로 예측하는 모델을 제작하였다.

---

## 실험 조건

| 변수            | 조건         |
| --------------- | ------------ |
| 효소 원료       | 감자         |
| 감자 질량       | 0.5 g        |
| 과산화수소 용액 | 15 mL        |
| 반응 시간       | 1분          |
| 측정값          | 반응 후 질량 |

산소 발생량을 직접 측정하기 어려웠기 때문에, 반응 후 질량 감소량을 효소 활성의 지표로 사용하였다.

---

## 실험 데이터

| pH  | 반응 후 질량(g) |
| --- | --------------- |
| 4   | 15.1            |
| 7   | 15.2            |
| 8   | 15.3            |
| 10  | 14.8            |

효소 활성도는 다음과 같이 정의하였다.

```text
Activity = m0 - m
```

- `m0`: 기준 질량
- `m`: 반응 후 측정 질량

즉, 질량 감소량이 클수록 더 많은 산소가 발생한 것으로 해석하였다.

---

## 데이터 증강(Data Augmentation)

원본 실험 데이터는 pH 종류가 4개뿐이었기 때문에 머신러닝 분석에 충분하지 않았다.

이를 보완하기 위해 다음 과정을 수행하였다.

### 1. 보간(Interpolation)

실험하지 않은 중간 pH 값을 선형 보간으로 생성하였다.

예:

- pH 4 ~ pH 7 사이의 중간값 생성

### 2. Monte Carlo 시뮬레이션

실험 오차를 가정하여 Gaussian Noise를 추가하였다.

사용한 식:

```text
y_sim = y_obs + ε
```

```text
ε ~ N(0, σ²)
```

설명:

- `y_sim`: 생성된 가상 데이터
- `y_obs`: 실제 실험 데이터
- `ε`: 랜덤 오차
- `N(0, σ²)`: 평균이 0이고 분산이 σ²인 정규분포

이를 통해 실제 실험에서 발생할 수 있는 측정 오차를 모사하였다.

---

## 머신러닝 모델

본 프로젝트에서는 2차 Polynomial Regression 모델을 사용하였다.

회귀식:

```text
y = ax² + bx + c
```

- `x`: pH
- `y`: 효소 활성도

이를 통해 연속적인 pH 범위에서 효소 활성을 예측하였다.
